# Yandex

所有与Yandex Cloud相关的功能

>[Yandex Cloud](https://cloud.yandex.com/en/) 是一个公共云平台。

## 安装和设置

可以通过 PyPI 使用 pip 安装 Yandex Cloud SDK：

```bash
pip install yandexcloud
```
