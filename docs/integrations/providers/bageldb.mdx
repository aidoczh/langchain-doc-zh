# BagelDB

> [BagelDB](https://www.bageldb.ai/)（`用于人工智能的开放向量数据库`），类似于 AI 数据的 GitHub。

这是一个协作平台，用户可以在其中创建、共享和管理向量数据集。它可以支持独立开发者的私人项目、企业内部的协作，以及数据 DAOs 的公共贡献。

## 安装和设置

```bash
pip install betabageldb
```
