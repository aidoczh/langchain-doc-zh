# Notion 数据库

[Notion](https://www.notion.so/) 是一个协作平台，支持修改过的 Markdown，并集成了看板、任务、维基和数据库。它是一个集合了笔记、知识和数据管理，以及项目和任务管理功能的多合一工作空间。

查看 [NotionDirectoryLoader 的使用示例](/docs/integrations/document_loaders/notion)。
