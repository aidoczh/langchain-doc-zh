# MindsDB

MindsDB 是一款用于定制企业数据人工智能的平台。借助 MindsDB 及其近 200 种与[数据源](https://docs.mindsdb.com/integrations/data-overview)和[AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview)的集成，任何开发人员都可以利用其企业数据快速、安全地定制人工智能。

使用 MindsDB，您可以将任何数据源连接到任何 AI/ML 模型，以实现并自动化 AI 动力应用。部署、提供服务并实时微调模型，利用来自数据库、向量存储或应用程序的数据。所有这些都可以使用开发人员已经熟悉的通用工具来完成。

MindsDB 与 LangChain 集成，使用户能够：

- 在 MindsDB 中部署通过 LangChain 可用的模型，使其可以访问众多数据源。

- 使用实时和动态数据在 MindsDB 中微调通过 LangChain 可用的模型。

- 使用 LangChain 和 MindsDB 自动化 AI 工作流程。

请参阅[我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain)，了解 MindsDB 与 LangChain 的集成，并查看示例。
