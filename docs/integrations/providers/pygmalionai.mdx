# PygmalionAI

[PygmalionAI](https://pygmalion.chat/) 是一家支持开源模型的公司，通过为[Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine)提供推理端点来支持开源模型。

## 安装和设置

```bash
pip install aphrodite-engine
```

## LLMs

查看[使用示例](/docs/integrations/llms/aphrodite)。

```python
from langchain_community.llms import Aphrodite
```
