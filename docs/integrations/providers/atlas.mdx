# Atlas

[Nomic Atlas](https://docs.nomic.ai/index.html) 是一个用于与小型和互联网规模的非结构化数据集进行交互的平台。

## 安装和设置

- 使用 `pip install nomic` 安装 Python 包

- `Nomic` 也包含在 langchains poetry extras 中，可以使用 `poetry install -E all` 安装

## VectorStore

查看 [使用示例](/docs/integrations/vectorstores/atlas)。

```python
from langchain_community.vectorstores import AtlasDB
```
