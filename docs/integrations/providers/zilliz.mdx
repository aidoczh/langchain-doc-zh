# Zilliz

[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是 LF AI Milvus® 在云端的全托管服务。

## 安装和设置

安装 Python SDK：

```bash
pip install pymilvus
```

## Vectorstore

Zilliz 索引的包装器允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Miluvs 包装器的更详细演练，请参阅[此笔记本](/docs/integrations/vectorstores/zilliz)。
