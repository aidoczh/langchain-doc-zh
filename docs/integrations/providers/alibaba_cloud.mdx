# 阿里巴巴云计算

>[阿里巴巴集团有限公司（维基百科）](https://zh.wikipedia.org/wiki/%E9%98%BF%E9%87%8C%E5%B7%B4%E5%B7%B4%E9%9B%86%E5%9B%A2)，简称 `阿里巴巴`，是一家中国的跨国科技公司，专注于电子商务、零售、互联网和技术领域。

>

>[阿里巴巴云计算（维基百科）](https://zh.wikipedia.org/wiki/%E9%98%BF%E9%87%8C%E4%BA%91)，也被称为 `阿里云`，是阿里巴巴集团的子公司。阿里巴巴云计算为在线业务和阿里巴巴自身的电子商务生态系统提供云计算服务。
