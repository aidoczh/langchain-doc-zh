# Groq

欢迎来到 Groq！🚀 在 Groq，我们开发了世界上第一款语言处理单元™，或者 LPU。Groq LPU 具有确定性、单核流式架构，为 GenAI 推理速度设定了标准，并且针对任何给定的工作负载具有可预测和可重复的性能。

除了架构之外，我们的软件旨在为像您这样的开发人员提供所需的工具，以创建创新的、强大的人工智能应用。有了 Groq 作为您的引擎，您可以：

* 实现实时人工智能和高性能计算推理的无损延迟和性能🔥

* 了解任何给定工作负载的确切性能和计算时间🔮

* 利用我们的尖端技术，保持领先地位💪

想要了解更多关于 Groq 吗？请访问我们的[网站](https://groq.com)获取更多资源，并加入我们的[Discord 社区](https://discord.gg/JvNsBDKeCG)与我们的开发人员联系！

## 安装和设置

安装集成包：

```bash
pip install langchain-groq
```

请求[API 密钥](https://wow.groq.com)并将其设置为环境变量：

```bash
export GROQ_API_KEY=gsk_...
```
