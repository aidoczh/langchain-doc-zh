# Stack Exchange

>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个涵盖多个领域话题的问答网站网络，每个网站涵盖一个特定主题，用户可以在这里进行提问、回答问题，并且会根据声誉奖励机制进行评定。

本页面介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装和设置

- 使用以下命令安装所需的库：

```bash
pip install stackapi
```

## 封装器

### 实用工具

存在一个名为 StackExchangeAPIWrapper 的实用工具，用于封装该 API。要导入此实用工具：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此封装器的更详细说明，请参阅[此笔记本](/docs/integrations/tools/stackexchange)。

### 工具

您还可以将此封装器轻松加载为工具（用于与代理一起使用）。

您可以使用以下代码实现：

```python
from langchain.agents import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参阅[此页面](/docs/how_to/tools_builtin)。
