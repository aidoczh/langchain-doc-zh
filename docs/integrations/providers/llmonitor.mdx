# LLMonitor

[LLMonitor](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs) 是一个开源的可观测性平台，提供成本和使用情况分析、用户跟踪、追踪和评估工具。

## 安装和设置

在 [llmonitor.com](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs) 上创建一个账户，然后复制你的新应用的 `tracking id`。

一旦你获得了它，通过以下命令将其设置为环境变量：

```bash
export LLMONITOR_APP_ID="..."
```
