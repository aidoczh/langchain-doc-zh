# Beam

[Beam](https://www.beam.cloud/) 是一个云计算平台，允许您在带有 GPU 的远程服务器上运行您的代码。

## 安装和设置

- [创建一个账户](https://www.beam.cloud/)

- 使用以下命令安装 Beam CLI：`curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh`

- 使用 `beam configure` 注册 API 密钥

- 设置环境变量 (`BEAM_CLIENT_ID`) 和 (`BEAM_CLIENT_SECRET`)

- 安装 Beam SDK：

```bash
pip install beam-sdk
```
