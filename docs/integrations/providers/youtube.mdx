# YouTube

[YouTube](https://www.youtube.com/) 是由 Google 推出的在线视频分享和社交媒体平台。

我们下载了 YouTube 的字幕和视频信息。

## 安装和设置

```bash
pip install youtube-transcript-api
pip install pytube
```

查看[使用示例](/docs/integrations/document_loaders/youtube_transcript)。
