# Meilisearch

> [Meilisearch](https://meilisearch.com) 是一款开源、速度极快且超相关的搜索引擎。

> 它具有出色的默认设置，可帮助开发人员构建迅捷的搜索体验。

您可以选择 [自行托管 Meilisearch](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation) 或在 [Meilisearch 云服务](https://www.meilisearch.com/pricing) 上运行。

`Meilisearch v1.3` 支持向量搜索。

## 安装与设置

查看[使用示例](/docs/integrations/vectorstores/meilisearch)以获取详细的配置说明。

我们需要安装 `meilisearch` Python 包。

```bash
pip install meilisearch
```
