# Argilla

[Argilla](https://argilla.io/) 是一个面向LLM的开源数据整理平台。

使用 `Argilla`，每个人都可以通过更快的数据整理，结合人工和机器反馈，构建健壮的语言模型。

`Argilla` 在MLOps周期的每个步骤都提供支持，从数据标注到模型监控。

## 安装和设置

获取你的 [API key](https://platform.openai.com/account/api-keys)。

安装 Python 包：

```bash
pip install argilla
```

## 回调

```python
from langchain.callbacks import ArgillaCallbackHandler
```

查看一个[示例](/docs/integrations/callbacks/argilla)。
