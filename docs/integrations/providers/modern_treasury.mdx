# 现代财务

[现代财务](https://www.moderntreasury.com/)简化复杂的支付操作。它是一个统一的平台，用于支持移动资金的产品和流程。

- 连接银行和支付系统

- 实时跟踪交易和余额

- 为规模化自动化支付操作

## 安装和设置

对此没有特殊的设置要求。
