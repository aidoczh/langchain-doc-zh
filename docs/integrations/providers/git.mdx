# Git

[Git](https://en.wikipedia.org/wiki/Git) 是一种分布式版本控制系统，用于跟踪任何一组计算机文件的更改，通常用于协调程序员在软件开发过程中共同开发源代码的工作。

## 安装和设置

首先，您需要安装 `GitPython` Python 包。

```bash
pip install GitPython
```

## 文档加载器

查看 [使用示例](/docs/integrations/document_loaders/git)。

```python
from langchain_community.document_loaders import GitLoader
```
