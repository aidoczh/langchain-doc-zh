# GitHub

>[GitHub](https://github.com/) 是一个开发者平台，允许开发者创建、存储、管理和分享他们的代码。它使用 `Git` 软件，提供了Git的分布式版本控制以及访问控制、bug跟踪、软件功能请求、任务管理、持续集成以及每个项目的维基。

## 安装和设置

要访问GitHub API，您需要一个[个人访问令牌](https://github.com/settings/tokens)。

## 文档加载器

GitHub有两种文档加载器可用。

查看一个[使用示例](/docs/integrations/document_loaders/github)。
