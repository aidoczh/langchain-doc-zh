# Typesense

> [Typesense](https://typesense.org) 是一个开源的内存搜索引擎，您可以选择自行

> [托管](https://typesense.org/docs/guide/install-typesense.html#option-2-local-machine-self-hosting)或在

> [Typesense 云](https://cloud.typesense.org/)上运行。

> `Typesense` 专注于性能，通过将整个索引存储在 RAM 中（同时备份到磁盘），并且专注于提供开箱即用的开发者体验，简化可用选项并设置良好的默认值。

## 安装和设置

```bash
pip install typesense openapi-schema-pydantic
```
