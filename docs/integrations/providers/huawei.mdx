# 华为

[华为技术有限公司](https://www.huawei.com/)是一家中国跨国数字通信技术公司。

[华为云](https://www.huaweicloud.com/intl/en-us/product/)提供全面的全球云计算服务套件。

## 安装和设置

要访问`华为云`，您需要一个访问令牌。

您还需要安装一个Python库：

```bash
pip install -U esdk-obs-python
```

## 文档加载器

### 华为OBS目录

查看[使用示例](/docs/integrations/document_loaders/huawei_obs_directory)。

```python
from langchain_community.document_loaders import OBSDirectoryLoader
```

### 华为OBS文件

查看[使用示例](/docs/integrations/document_loaders/huawei_obs_file)。

from langchain_community.document_loaders.obs_file import OBSFileLoader
