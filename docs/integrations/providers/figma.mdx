# Figma

[Figma](https://www.figma.com/) 是一个用于界面设计的协作 Web 应用程序。

## 安装和设置

Figma API 需要 `access token`、`node_ids` 和 `file key`。

`file key` 可以从 URL 中获取。https://www.figma.com/file/{filekey}/sampleFilename

`Node IDs` 也可以在 URL 中找到。单击任何内容，查找 '?node-id={node_id}' 参数。

`Access token` 的 [使用说明](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)。
