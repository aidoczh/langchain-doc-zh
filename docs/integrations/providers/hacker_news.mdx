# 黑客新闻

>[黑客新闻](https://en.wikipedia.org/wiki/Hacker_News)（有时缩写为 `HN`）是一个专注于计算机科学和创业精神的社交新闻网站。它由投资基金和初创企业孵化器 `Y Combinator` 运营。一般来说，可以提交的内容被定义为“满足人们的求知欲的任何内容”。

## 安装和设置

对于它并没有特别的设置要求。
