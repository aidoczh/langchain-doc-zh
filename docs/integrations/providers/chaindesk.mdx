# Chaindesk

[Chaindesk](https://chaindesk.ai) 是一个开源文档检索平台，可以帮助将个人数据与大型语言模型连接起来，访问其[GitHub](https://github.com/gmpetrov/databerry)。

## 安装和设置

我们需要注册 Chaindesk，创建数据存储，添加一些数据，并获取数据存储的 API 端点 URL。我们需要[API 密钥](https://docs.chaindesk.ai/api-reference/authentication)。
