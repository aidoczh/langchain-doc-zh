# Trubrics

[Trubrics](https://trubrics.com) 是一个LLM用户分析平台，可以帮助您收集、分析和管理对AI模型的用户提示和反馈。

查看[Trubrics存储库](https://github.com/trubrics/trubrics-sdk)以获取有关`Trubrics`的更多信息。

## 安装和设置

我们需要安装`trubrics` Python包：

```bash
pip install trubrics
```

## 回调

查看[用法示例](/docs/integrations/callbacks/trubrics)。

```python
from langchain.callbacks import TrubricsCallbackHandler
```
