# AINetwork

[AI Network](https://www.ainetwork.ai/build-on-ain) 是一个设计用于容纳大规模人工智能模型的一级区块链，利用由 [$AIN token](https://www.ainetwork.ai/token) 驱动的分散式 GPU 网络，丰富了基于人工智能的 `NFTs` (`AINFTs`)。

## 安装和设置

您需要安装 `ain-py` Python 包。

```bash
pip install ain-py
```

您需要将 `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` 环境变量设置为您的 AIN 区块链账户私钥。

## 工具包

查看 [使用示例](/docs/integrations/toolkits/ainetwork)。

```python
from langchain_community.agent_toolkits.ainetwork.toolkit import AINetworkToolkit
```
