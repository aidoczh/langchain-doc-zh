# Cube

>[Cube](https://cube.dev/) 是用于构建数据应用程序的语义层。它帮助数据工程师和应用程序开发人员从现代数据存储中访问数据，将其组织成一致的定义，并将其传递给每个应用程序。

## 安装和设置

我们需要获取 Cube 实例的 API 密钥和 URL。请参阅[这些说明](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path)。
