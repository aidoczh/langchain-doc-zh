# SerpAPI

本页面介绍如何在 LangChain 中使用 SerpAPI 搜索 API。内容分为两部分：安装和设置，以及特定 SerpAPI 包装器的参考信息。

## 安装和设置

- 使用 `pip install google-search-results` 安装所需的依赖项

- 获取 SerpAPI API 密钥，并将其设置为环境变量 (`SERPAPI_API_KEY`)

## 包装器

### 实用工具

存在一个包装了该 API 的 SerpAPI 实用工具。要导入此实用工具：

```python
from langchain_community.utilities import SerpAPIWrapper
```

有关此包装器的更详细的步骤，请参阅[此笔记本](/docs/integrations/tools/serpapi)。

### 工具

您还可以将此包装器轻松加载为工具（用于与代理一起使用）。您可以这样做：

```python
from langchain.agents import load_tools
tools = load_tools(["serpapi"])
```

有关此内容的更多信息，请参阅[此页面](/docs/how_to/tools_builtin)。
