# Joplin

[Joplin](https://joplinapp.org/) 是一款开源的笔记应用。它可以记录你的想法，并且可以在任何设备上安全地访问这些记录。

## 安装和设置

`Joplin API` 需要一个访问令牌。你可以在[这里](https://joplinapp.org/api/references/rest_api/)找到安装说明。

## 文档加载器

查看一个[使用示例](/docs/integrations/document_loaders/joplin)。

from langchain_community.document_loaders import JoplinLoader
