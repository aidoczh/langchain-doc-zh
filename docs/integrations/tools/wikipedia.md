# 维基百科

[维基百科](https://wikipedia.org/)是一个由志愿者社区撰写和维护的多语言免费在线百科全书，这些志愿者被称为维基百科人，通过开放协作并使用名为MediaWiki的基于维基的编辑系统。`维基百科`是历史上最大和最受欢迎的参考作品。
