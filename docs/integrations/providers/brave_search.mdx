# Brave Search

[Brave Search](https://en.wikipedia.org/wiki/Brave_Search) 是由 Brave Software 开发的搜索引擎。

- `Brave Search` 使用自己的网络索引。截至 2022 年 5 月，它覆盖了超过 100 亿个页面，并且在 92% 的搜索结果中不依赖任何第三方，其余的结果是从必应 API 服务器端检索或（根据用户选择）从 Google 客户端检索。根据 Brave 公司的说法，索引被故意保持“比 Google 或必应小”，以帮助避免垃圾信息和其他低质量内容，但缺点是“Brave Search 在恢复长尾查询方面还不如 Google”。

- `Brave Search Premium`：截至 2023 年 4 月，Brave Search 是一个无广告的网站，但最终将转向一个新模式，其中将包括广告，而高级用户将享有无广告体验。用户数据，包括 IP 地址，不会默认收集。高级账户将需要用户选择加入数据收集。

## 安装和设置

要访问 Brave Search API，您需要[创建一个账户并获取 API 密钥](https://api.search.brave.com/app/dashboard)。
