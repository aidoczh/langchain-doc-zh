# 电报

[Telegram Messenger](https://web.telegram.org/a/) 是一款全球可访问的免费即时通讯服务，支持跨平台、加密、基于云的集中式架构。该应用还提供可选的端到端加密聊天和视频通话、VoIP、文件共享以及其他几项功能。

请参阅[设置说明](/docs/integrations/document_loaders/telegram)。
