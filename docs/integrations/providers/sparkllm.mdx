# SparkLLM

[SparkLLM](https://xinghuo.xfyun.cn/spark) 是讯飞自主开发的大规模认知模型。它通过学习大量的文本、代码和图片，具备跨领域知识和语言理解能力，能够基于自然对话理解并执行任务。
