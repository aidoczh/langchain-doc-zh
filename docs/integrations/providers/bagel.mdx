# 贝格尔

> [贝格尔](https://www.bagel.net/)（`面向人工智能的开放向量数据库`），就像是面向人工智能数据的 GitHub。

这是一个协作平台，用户可以在其中创建、分享和管理向量数据集。它可以支持独立开发者的私人项目、企业内部的协作，以及数据 DAOs 的公共贡献。

## 安装和设置

```bash
pip install bagelML
```

## 向量存储

查看[使用示例](/docs/integrations/vectorstores/bagel)。

from langchain_community.vectorstores import Bagel
