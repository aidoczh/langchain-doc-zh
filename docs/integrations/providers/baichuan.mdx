# 百川

[百川公司](https://www.baichuan-ai.com/)是一个中国初创公司，致力于在通用人工智能时代解决人类的基本需求：效率、健康和幸福。

## 安装和设置

在[这里](https://platform.baichuan-ai.com/)注册并获取 API 密钥。

## 语言模型

查看[使用示例](/docs/integrations/llms/baichuan)。

```python
from langchain_community.llms import BaichuanLLM
```

## 聊天模型

查看[使用示例](/docs/integrations/chat/baichuan)。

```python
from langchain_community.chat_models import ChatBaichuan
```

## 嵌入模型

查看[使用示例](/docs/integrations/text_embedding/baichuan)。

from langchain_community.embeddings import BaichuanTextEmbeddings
